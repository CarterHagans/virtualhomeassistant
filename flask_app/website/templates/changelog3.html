AI adaptability and recognition: 6-1-23

  • ⇨ AI Adaptability: After receiving feedback from users, the AI has now learned a lot of new vocabulary in order for easier use of commands in the texting application of the app. You are now able to say things such as “add an item to the list” or “edit an item on the list” or “who did that”, or “how do i see my families”, “join my friends family”, “how to see the shopping list”, “who are the admins of this family”, “i need to change family details”, and many more options for you to be able to have an easy access to commands!
  • ⇨ AI error messages. The bot will now give you feedback when something goes wrong in the usage of a command. For a lot of commands, you must be logged into a family to be able to execute those commands. The bot will alert you if you are not logged into a family so that you know you have to be logged in.
  • ⇨ Change log added to the application. Whenever there is an update to the application publicly, there will be a text notification sent out. Along with the text notifications, you will be able to access the change logs with version numbers of the application at https://virtualshoppinglist.com/changelog
